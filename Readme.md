# Schattenlibary für Wasdazu Webseite

# Benützung

## Abhebung eines HTML-Elementes vom Rest der Seite

```
<div class="shadow-strongest">
    <h2>Berne</h2>
    <p>Berne is the capital of Switzerland.</p>
</div>```

## Abhebung eines HTML-Elementes und Hintergrundfarbe ändern
```
<div class="shadow-strongest wasdazu-orange">
    <h2>Berne</h2>
    <p>Berne is the capital of Switzerland.</p>
</div>```
